Browser Operation

To perform given operations use browser console
Type the command in browser console by
RightClick-> inspect -> console
Prime Number ex: primenumber(2)
Fibonacci ex: fibonacci(5)
Strong Number ex: strongNumber(123)
Armstrong ex: armstrong(5)
Factorial ex: Factorial(5)
Temperature conversion F -> C ex: tempConversion(300)
Output Display

Command used
on browser console: primenumber(4)

4 is not a prime number
